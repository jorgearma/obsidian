# Ejercicio 3 — Connector Flow

Objetivo: aprender a **extender ideas automáticamente** usando conectores naturales.

Esto es lo que diferencia un **B2 de un C1** al hablar.

---

# Base del método

Se toma **una idea simple** y se obliga a expandirla usando conectores.

Ejemplo idea base:

> I like working on my own.

Ahora se expande.

---

# Paso 1 — idea simple

Frase básica.

Example:

> I like working on my own.

---

# Paso 2 — expansión con conector

Añade **because / since / as**.

Example:

> I like working on my own because I can organize my time.

---

# Paso 3 — contraste

Añade **although / even though / but**.

Example:

> I like working on my own because I can organize my time, even though sometimes it can be stressful.

---

# Paso 4 — consecuencia

Añade **so / that's why / which means**.

Example:

> I like working on my own because I can organize my time, even though sometimes it can be stressful, so I try to plan my day carefully.

---

# Paso 5 — reflexión

Añade **actually / in fact / the truth is**.

Example:

> Actually, the truth is that I prefer this lifestyle because it gives me freedom.

---

# Ejemplo completo

Idea inicial:

> I like working on my own.

Resultado final:

> I like working on my own because I can organize my time, even though sometimes it can be stressful, so I try to plan my day carefully. Actually, the truth is that I prefer this lifestyle because freedom is very important to me.

Esto ya es **fluidez natural**.

---

# Cómo practicar (10 minutos)

Toma una idea simple y expándela.

Ejemplo de ideas:

I like living in Denmark.  
I enjoy learning English.  
I want to work in cybersecurity.  
I don't like working under pressure.  
I prefer freedom over stability.

Expande cada una hasta **5–6 frases conectadas**.
